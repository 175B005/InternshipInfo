## 2018/1/23 [CLINKS株式会社](http://www.clinks.jp/)  

1dayインターンシップに参加。派遣会社で中小。社長が株を100%持っている。  
決定が早く、社内でのイベントをすることで、派遣による社員の疎遠などは防げるらしい。  
社内SNSがあって、まあ、連絡とか、進行とか、報告とか  
そういうのまで一括にやっているものではあった。  
チームで活動するらしく、今の専門学校と同じ感じになっていると思った。  
今回は説明というか、感じを掴むためということで、概要説明だったので、  
軽い模擬面接をさせていただきましたが、まあまあという感じだったので、  
次に生かせるようにしたいですね。。

####  見た企業

- [ランド・ホー](https://www.landho.co.jp/)  

番組制作、委託開発、もともとコンシューマゲームで、今それをオンライン、スマホに起こしている  
技術面ではしっかり、でも、上場には敵わない感じ、、他との連携開発多め。

- [DeNA](http://dena.com/jp/company/)  

コアスキル重要。技術重視。エリート思考。向上のための模索、チャレンジは歓迎。  
やりたいこと、必要となる仕事などに力をいれ、そのための設備は全て用意する。  
→資本があり、多くの事業展開をしている。野球など、ゲームだけでなく、リアルの  
アミューズメント、社会の促進など、先端に立つ感じ。  
ただ、やることが人によって違うから、研修充実というよりは、  
自主学習の完全サポート、という感じで、外部の講習会など、お金を出してくれる、らしい。  

- [ワンダープラネット](http://wonderpla.net/)  

360度開放を掲げている、上場ではないので、やはり自社開発というよりか、  
他の企業委託、連携などで活躍。株なども他が持っている？  
→自由にできないところもあるらしい。でも、活動としては幅広く、  
技術面なども信頼のある企業として、今後活躍していく会社。。かな？  
模倣が多いと感じた。自由にできないというよりかは、  
形式に自らとらわれている感じ。理念も、向かっているが、いま一歩  
できていない、、というニュアンスが感じ取れた。

- [ヘキサドライブ](https://hexadrive.jp/)  

ＨＤ系のグラフィックがすご。。  
リメイクやその他連携として、技術提供している会社。  
グラフィッカーがメインかな？

- [カプコン](http://www.capcom.co.jp/)  

言わずもがなモンハンの話、その他有名タイトル多数。  
ただし、採用って感じではなく、やっぱ経験者求めてるよなって感じる説明内容。  
自社ＩＰをフル活用。一つの物を完璧に作り上げ、それをバンバン流用していくスタイル  
売れたものは優遇だが、知らんタイトルはほんとに知らん。 
入って、有名なものに携われるかって言ったら、そうではないだろう。  
他にも力を入れていることが多いが、それは経験者しかできないでしょ。  
規模が違う。

- [テクロス](https://www.techcross.co.jp/)

独自ＩＰ持ち、ソーシャルゲーム急成長。

unity などの既存ゲームエンジンにて、開発運用を行っている。

独創があまりない。アイデアや社会に対応するのでいっぱいいっぱい。

- [コロプラ](http://colopl.co.jp/index.php)  

いろんな視点で、一つ一つちょっとやってみる、からはいる。
tryに力をいれ、出来そうなこと、やりたいことを早い段階で始める。
イメージ型→　判断、意見交換→　α版→　β版→　完成
基本人数は２、３人で軽いノリで始めて、
提案だして、基本の動きなんかを作ったら、皆に発表。
通れば大人数になっていって、完成。
基本フレームに追加していくという、今のソーシャルゲームの
考え方そのものを会社全体で行う。
プランナーでもプログラムしたり、口出したり、
仲間内というノリが大事、コミュニケーション重要。→孤立したらやばい。。。とおもた。。。
アセットの作成や、少人数の組み合わせで大きなチームになるとか、
細かいフレーム単位で完成させ、財産にしてくスタイル。
新規開発が、あまりうまくいかないのかもしれない。。（アイデア面）
新たなプラットフォームにも力をいれ、ＶＲ、アーケードなど、
展開を拡大しようとしている時期。つまり、成長が止まり気味かもしれない。
unity unrealといっていたので、それもいつまでかわからないし、急にばたっと
なるかもしれないという危惧が強く感じられる。
まあ、これまでも、業界内では型やぶりに突き進んできたところがあるので、
そこは信頼できるところ。スマホが普及してからすぐの、市場をつかむセンスと、
先駆けの精神は、私が目指す理想の考えそのもの（この業界に対する）。
なんか特許でもめ事があったが、、どうなのだろうか。
偏った技術ではあると思うんだよね。今後どうなるか。。。
採用→エンジニアとして採用枠はあるが、ゲームエンジンなら、プログラマいらなくね？
デザイナーとか、力入れたいのは表現演出、今は３Ｄモデラ―ではないかと思うんだよねー。
やっぱり、ある技術を使わないとだしね。
箱根に温泉→資本があるし、今は安定期、新規開発のため、
いろいろと模索してるんだなーというのがすごく分かる。
こうやって、しっかりとどうすればいいか、そして、やってみるってことを
ちゃんとできているので、その点、衰退とかは考えられないね。。。
今欲しいのは、新しいことできる人なのかなー。
俺にピッタリやね！、、、、はい、すいません。。
でも、考えてみるとそうなんよ、今取り掛かっていることは、maya unity 連携機能紹介のやつで、
あれは思いっきし新しいことだし、今日の話でも、
プロジェクタに写っていた工程は、全部俺が紹介しているもの。。
縁を感じてならない。。「あ、同じことやってる。できる。」と思た。
会社に入れば、メモリとかの関係で、効率にやっぱり気を付けることになると思うけど、
入って即戦力になれるなーと思った。
でもね、、３ｄモデラ―の分野って、デザイナーやん。
エンジニアじゃないねん。絵描けない。。デザイナー目指すか、、
垣根ないって言ってたからといって、そういうことでもないと思うんだよね。
まあ、とりあえず、今日ここが第一志望になりましたわ。
