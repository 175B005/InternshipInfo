## 2018/1/23 [CLINKS株式会社](http://www.clinks.jp/)  

1dayインターンシップに参加。派遣会社で中小。社長が株を100%持っている。  
決定が早く、社内でのイベントをすることで、派遣による社員の疎遠などは防げるらしい。  
社内SNSがあって、まあ、連絡とか、進行とか、報告とか  
そういうのまで一括にやっているものではあった。  
チームで活動するらしく、今の専門学校と同じ感じになっていると思った。  
今回は説明というか、感じを掴むためということで、概要説明だったので、  
軽い模擬面接をさせていただきましたが、まあまあという感じだったので、  
次に生かせるようにしたいですね。。
